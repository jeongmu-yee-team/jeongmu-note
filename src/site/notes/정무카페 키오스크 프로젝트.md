## 정무카페 키오스크 프로젝트
### 프로젝트 목적
+ React 프로젝트를 진행함으로 웹 개발 및 웹 디자인 능력을 함양한다.
+ 웹 디자인 설계 및 개발, 배포함으로써 전반적인 웹 프로젝트를 진행해본다.

### 프로젝트 소개
+ 이 프로젝트는 청운대학교 "정무2팀"에서 진행하는 웹 개발 프로젝트입니다.
+ 본 프로젝트는 React App에 대한 학습 및 연습에 중점을 둔 프로젝트입니다.
